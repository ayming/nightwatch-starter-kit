# Nightwatch starter kit
Simplify the configuration of nightwatch

## Requirements
* java sdk `~1.8`
* node `^4.5.0`
* yarn `^0.17.0` or npm `^3.0.0`

## Getting Started
```bash
$ yarn install    # Install project dependencies
$ yarn start      # Compile and launch (same as `npm start`)
```
